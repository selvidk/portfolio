Website Pemesanan Undangan

Project aplikasi berbasis web untuk pemesanan undangan yang dilengkapi dengan fitur edit teks pada template undangan dan dilengkapi dengan payment gateway untuk user calon pembeli. Untuk sisi admin dilengkapi dengan fitur export laporan dalam format excel dan pdf.

Sumber Daya Proyek
  • Framework Laravel 8.83.27
  • HTML, CSS, dan JS
  • Template Bootstrap 5, Startbootsrap SB Admin 2 dan eLearning HTML Template
  • Payment gateway Midtrans
  • MySQL
